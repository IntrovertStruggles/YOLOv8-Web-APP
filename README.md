# YOLOv8-Web-APP
Application of YOLOv8-DeepSORT algorithm in the form of a local web app with the help of StreamLit

the DeepSORT Algorithm fork used in this comes from https://github.com/computervisioneng/object-tracking-yolov8-deep-sort
YOLOv8 provided by Ultralytics

requires Python 3.7 or above
